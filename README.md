# Bauhaus Design Collective

A static, multi-page demo website with a **Bauhaus-inspired design theme** — geometric shapes, primary colors, bold typography, and hard edges (no rounded corners). Built with plain HTML, CSS, and vanilla JavaScript.

**Live site:** https://mbilalsiddiqi.github.io/bauhaus_web1/

## Pages
- `index.html` — The Manifesto (landing)
- `gallery.html` — The Works (gallery grid)
- `contact.html` — The Connection (contact form)

## Tech
- HTML5, CSS3 (custom properties, Grid, Flexbox), vanilla JavaScript
- Google Fonts: Archivo Black + Inter
- Shared `styles.css` and `script.js`

## Run locally
No build step required — open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000   # visit http://localhost:8000
```
